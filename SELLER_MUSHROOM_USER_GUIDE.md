# 卖家菌菇种类功能使用说明

## 功能概述

卖家菌菇种类管理功能允许卖家在注册时选择要售卖的菌菇种类，并在后续管理中查看和申请添加新的菌菇种类。

## 使用流程

### 1. 卖家注册时选择菌菇种类

**步骤：**
1. 访问注册页面（`/register`）
2. 选择角色为"卖家"
3. 填写用户名、密码、邮箱、手机号等信息
4. 在"菌菇种类"下拉框中选择要售卖的菌菇种类（可多选）
5. 阅读并同意《卖家服务协议》和《隐私政策》
6. 点击"注册"按钮

**注意事项：**
- 至少选择一种菌菇种类
- 选择的菌菇种类会在注册成功后自动关联到卖家账号
- 状态默认为"已批准"，无需管理员审核

### 2. 卖家查看自己的售卖种类

**步骤：**
1. 以卖家身份登录系统
2. 访问"我的售卖种类管理"页面（`/seller-mushrooms`）
3. 页面会显示所有已批准的菌菇种类
4. 每个菌菇卡片显示：图片、名称、分类

**功能：**
- 查看当前可以售卖的所有菌菇种类
- 点击"申请添加售卖种类"按钮可以申请新的菌菇种类

### 3. 卖家申请添加新的菌菇种类

**步骤：**
1. 在"我的售卖种类管理"页面点击"申请添加售卖种类"按钮
2. 选择申请方式：
   - **从现有列表选择**：从系统中已有的菌菇种类中选择
   - **填写新菌菇种类**：填写新的菌菇种类信息（名称、分类、描述）
3. 提交申请
4. 等待管理员审核

**审核状态：**
- `pending`：待审核
- `approved`：已批准（可以售卖）
- `rejected`：已拒绝

### 4. 管理员查看卖家的售卖种类

**步骤：**
1. 以管理员身份登录系统
2. 访问"用户管理"页面（`/admin/users`）
3. 找到卖家用户
4. 点击"查看售卖种类"按钮
5. 弹窗显示该卖家的所有已批准的菌菇种类

**功能：**
- 查看所有卖家的售卖种类
- 了解每个卖家的业务范围
- 便于管理和监督

### 5. 管理员审核卖家的菌菇种类申请

**步骤：**
1. 以管理员身份登录系统
2. 访问"卖家售卖种类审核"页面（`/admin/seller-mushroom-approval`）
3. 查看待审核的申请列表
4. 点击"通过"或"拒绝"按钮进行审核

**审核标准：**
- 菌菇种类是否符合平台规范
- 卖家是否有资质售卖该菌菇
- 是否存在重复申请

## API端点说明

### 1. 选择菌菇种类（注册时使用）
```
POST /api/seller-mushrooms/select
请求体: {
  "sellerId": 3,
  "mushroomIds": [6, 7, 8]
}
响应: {
  "success": true,
  "message": "菌菇种类选择成功",
  "count": 3
}
```

### 2. 获取卖家的菌菇种类列表
```
GET /api/seller-mushrooms/seller/:sellerId
Headers: {
  "Authorization": "Bearer <token>"
}
响应: {
  "success": true,
  "data": [
    {"id": 6, "name": "香菇", "image": "...", "category": "食用菇"},
    ...
  ],
  "count": 3
}
```

### 3. 申请添加新的菌菇种类
```
POST /api/seller-mushrooms/apply
Headers: {
  "Authorization": "Bearer <token>"
}
请求体: {
  "sellerId": 3,
  "mushroomId": 9  // 从现有列表选择
}
// 或者
请求体: {
  "sellerId": 3,
  "newMushroom": {
    "name": "新菌菇",
    "category": "食用菇",
    "description": "描述..."
  }
}
响应: {
  "success": true,
  "message": "申请已提交，等待管理员审核"
}
```

### 4. 管理员审核申请
```
POST /api/seller-mushrooms/approve
Headers: {
  "Authorization": "Bearer <token>"
}
请求体: {
  "id": 1,
  "approved": true  // true=通过, false=拒绝
}
响应: {
  "success": true,
  "message": "审核通过"
}
```

## 常见问题

### Q1: 为什么我注册时选择的菌菇种类没有保存？
**A:** 请检查以下几点：
1. 是否选择了至少一种菌菇种类
2. 注册时是否出现错误提示
3. 查看后端日志确认是否有错误信息
4. 联系管理员检查数据库中的seller_mushrooms表

### Q2: 为什么我看不到自己选择的菌菇种类？
**A:** 可能的原因：
1. 未正确登录（确保使用卖家账号）
2. 菌菇种类状态不是"approved"
3. 浏览器缓存问题，尝试刷新页面
4. 检查浏览器控制台是否有错误信息

### Q3: 如何添加新的菌菇种类到系统？
**A:** 有两种方式：
1. **卖家申请**：卖家在"我的售卖种类管理"页面申请添加，等待管理员审核
2. **管理员添加**：管理员直接在数据库的mushrooms表中添加新记录

### Q4: 管理员在哪里审核卖家的申请？
**A:** 管理员登录后访问"卖家售卖种类审核"页面（`/admin/seller-mushroom-approval`），可以看到所有待审核的申请。

### Q5: 一个卖家可以选择多少种菌菇？
**A:** 目前没有数量限制，卖家可以根据需要选择多种菌菇种类。
